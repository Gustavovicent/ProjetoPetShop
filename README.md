Este projeto esta sendo desenvolvido para solucionar a demanda de serviços de um pet-Shop com as opções de cadastro de clientes que esta
relacionado com o cadastro de animal onde o mesmo pode estar em varios serviços conectado ao SQL como banco de dados.
Tambem vou adicionar a opção de agenda para controlar os serviços e horarios agendados, Coom o tempo eu pretendo acrscentar mais APIS para 
o projeto ficar mais completo, como a digitalização de exames.
